PRE-REQUISITES

While this guide intends to be a beginners guide to setting up and using the Google Custom Search API within a Google App Engine driven app (also extends to other cloud or mobile based offerings), there are some basic requirements this guide will not cover. You should review this list to ensure that you are setup and familiar with the required skills/technologies before continuing on with this guide.

Google App Engine/SDK

This guide will explore setting up the Custom Search Engine associated with a Python based Google App Engine(GAE) website. It will not however cover the setup of the GAE SDK and Development environment. For a good guide on these topics please reference the links below:

SDK Download - https://cloud.google.com/appengine/downloads
GAE App Tutorial - https://cloud.google.com/appengine/docs/python/

Python, JSON, HTTP Requests

The GAE environment we will work with for this tutorial uses Python, and you should have a basic understanding of working with JSON structured data, and getting/receiving HTTP requests. Links for these topics provided below:

Introduction to Python - https://www.python.org/about/gettingstarted/
JSON with Python - https://pymotw.com/2/json/
HTTP with Python - https://docs.python.org/3/library/http.client.html#

HTML, webapp2, jinja2

While GAE apps can be built in different ways, to understand the source code for this project a basic understanding of HTML, the webapp2 and jinja2 libraries would be helpful. Below are some links covering the basics.

HTML Tutorial - http://www.w3schools.com/html/
webapp2 Documentation - https://webapp-improved.appspot.com/
jinja2 Documentation - http://jinja.pocoo.org/

There are of course many more resources available for any and all of these topics online. With this background you should be able to easily follow the upcoming guide. Let's get started with enabling the Google Custom Search API with our GAE app.
